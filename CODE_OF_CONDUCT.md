# Code of Conduct

We follow the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## Short version

Be kind. Be patient. Assume good faith. Ask questions before assuming
incompetence. Don't make this about politics, identity, or anything else
unrelated to the project.

## Reporting

Email **info@casperjuel.dk** privately. Reports stay between you and the
maintainer unless you explicitly want them public. We take harassment, abuse,
and discrimination seriously, and so should you.

## Enforcement

Maintainer (Casperjuel) decides on responses to reports. Range:
- Quiet word
- Public clarification
- Temporary block
- Permanent ban

For severe or repeat behaviour, expect the harsher options.

## Scope

This applies in all project spaces: issues, PRs, discussions, the Aula MCP
discord channel (when one exists), commits, code comments, and anywhere
people interact under the project's umbrella.
